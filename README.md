# Getting Started
## Installation Instructions
1. Clone this repository: 
   ```bash
   git clone https://github.com/Narek889/Reinforcement-Learning.git
   cd Reinforcement-Learning
   ```
2. Install dependencies: 
   ```bash
   pip install -r requirements.txt
   ```
